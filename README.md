### 插件功能

- 学者网
  - 点击[我的课程](https://www.scholat.com/myCourses.html)自动跳转到"学习的课程"
  - 自定义出现在课程页面"正在学习"的课程
- SSO平台
  - 去除公告栏
  - 默认显示"我的应用"而不是"常用应用"
- 教务系统
  - 自动跳过开头需要等待5s的界面
- 砺儒云
  - 首页显示课程
